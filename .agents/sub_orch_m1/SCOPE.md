# Scope: Milestone 1 - Data Cleaning

## Architecture
- `cafes.tsv` and `bakeries.tsv` currently contain 214 fake entries generated by `import_bookmarks.py`.
- We need to identify these fake entries and remove them, leaving exactly 30 validated entries combined (or however many initial ones there were).
- The fake entries can be identified because their columns match the randomly generated strings in `import_bookmarks.py` (e.g., `AMBIANCES`, `CAFE_SIG_MENUS`, etc.).

## Milestones
| # | Name | Scope | Dependencies | Status |
|---|------|-------|-------------|--------|
| 1 | Data Cleaning | Write a script to identify and remove fake entries in TSV files, leaving only the validated ones. | none | PLANNED |

## Interface Contracts
- Cleaned TSV files must keep the original header.
- Cleaned TSV files must have no fake entries.
