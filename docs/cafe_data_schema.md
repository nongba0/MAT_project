# 카페 데이터베이스 스키마 (Cafe Data Schema)

이 문서는 `cafes.tsv` 파일에 저장되며 Spring Boot 백엔드 애플리케이션의 `CafeSeeder.java`를 통해 파싱되는 카페 데이터 컬럼 17가지에 대한 명세서입니다. AI 기반의 맞춤형 장소 추천 및 비교를 위해 수집되는 메타데이터 구조를 정의합니다.

> [!NOTE]
> 빵집(`bakeries.tsv`)의 경우 컬럼 수가 14개로 카페 데이터베이스 구조와 다소 차이가 있으니 파싱 시 유의해야 합니다.

## 데이터 구조 명세서

모든 컬럼은 **탭(`\t`)** 문자로 구분되며, 총 17개의 컬럼이 존재합니다.

### 📍 기본 & 위치 정보
1. **상호명 (`name`)**: 카페의 공식 명칭 (예: `스타벅스 강남점`)
2. **정확한 주소 (`exact_address`)**: 도로명 주소 등 상세 주소
3. **위도 (`latitude`)**: 카카오맵/네이버 지도 API 연동 등을 위한 좌표 (예: `37.498095`)
4. **경도 (`longitude`)**: (예: `127.027610`)

### ☕ 메뉴 & 분위기 정보
5. **분위기 (`ambiance`)**: 인테리어, 공간 무드 등에 대한 서술적 묘사 (예: `통창 뷰의 조용한 감성 카페`)
6. **시그니처 메뉴 (`signature_menu`)**: 방문 시 꼭 먹어봐야 할 대표 메뉴 (예: `피넛크림 라떼, 바나나 푸딩`)
7. **메뉴 특징 (`menu_features`)**: 판매하는 음료 및 디저트의 퀄리티, 맛, 재료 등의 상세 설명

### 💬 내돈내산 리뷰 & 평가
8. **긍정적 리뷰 (`positive_reviews`)**: 실제 방문객(블로거, 유튜버 등)이 언급한 장점. 문장이 여러 개일 경우 파이프(`|`)로 구분합니다.
9. **부정적 리뷰 / 단점 (`negative_review`)**: 방문 시 아쉬웠던 점이나 주의할 점 (예: `화장실이 외부에 있어 다소 불편합니다.`)
10. **리뷰 링크 (`review_links`)**: 실제 데이터를 수집 및 교차 검증한 원본 리뷰 출처 URL. 여러 개일 경우 파이프(`|`)로 구분합니다.

### ⚙️ 편의 시설 & 이용 정보 (CafeOperation 엔티티)
11. **웨이팅 시스템 (`waiting_system`)**: 대기 방식 (예: `캐치테이블, 주말 오후 현장 대기 필수, 웨이팅 없음`)
12. **콘센트 유무 (`has_plugs`)**: 작업(카공)하기 좋은 환경인지 여부. `True` 또는 `False` 값만 허용.
13. **주차 정보 (`parking_info`)**: 전용 주차장 구비, 인근 공영주차장 이용, 주차 불가 등 구체적인 주차 난이도 설명
14. **디카페인 유무 (`has_decaf`)**: 디카페인 원두 변경이나 음료 옵션 존재 여부. `True` 또는 `False`.
15. **대형 카페 여부 (`is_large_cafe`)**: 매장 규모가 크고 좌석이 여유로운 대형 베이커리/카페인지 여부. `True` 또는 `False`.
16. **반려동물 동반 가능 (`is_pet_friendly`)**: 반려동물(주로 소형견) 동반 출입이 가능한지 여부. `True` 또는 `False`.

### 📊 AI 타겟 비교 데이터 (CafeComparison 엔티티)
17. **비교군 데이터 (`comparisons`)**: 사용자의 이해를 돕기 위해 잘 알려진 다른 브랜드나 장소와 직관적으로 비교하는 항목입니다. 다음과 같은 포맷을 **엄격히** 준수해야 합니다.
   * **포맷**: `[비교대상]|[비교특징]|[상세설명]`
   * **예시**: `스타벅스|카공 분위기|스타벅스가 일률적이고 붐비는 대형 작업 공간이라면, 이곳은 나만 알고 싶은 아지트 감성의 조용한 작업 공간`

> [!IMPORTANT]  
> 모든 데이터를 수집할 때는 광고성 정보가 아닌 **실제 내돈내산 리뷰**만을 활용하여 데이터를 구성해야 하며, `comparisons` 컬럼의 파이프(`|`) 포맷팅 오류는 런타임에 `CafeSeeder` 에러를 발생시킬 수 있으므로 각별히 주의해야 합니다.
